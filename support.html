Support

Last updated: February 2026

Need help? Email support@grammarspy.net.

We typically respond within 1–2 business days.

Fast Troubleshooting

  • Page looks old: hard refresh with Ctrl+Shift+R (or Cmd+Shift+R on Mac).
  • Class code issue: ask the teacher to confirm the exact code and try again with no spaces.
  • Sign-in issue: verify email + password, then try opening in a private window.
  • No sound: click once in the page to enable audio, then check your browser sound settings.
  • Game not loading: disable aggressive ad/script blockers for grammarspy.net and reload.

Teacher Help

For teacher setup support, include these details in your email:

  • Your account email
  • The page URL where the issue happened
  • A screenshot (if possible)
  • What you expected and what happened instead

Student Help

If students cannot join, share the exact class code and remind them to type letters and numbers exactly as shown.

Students should use the Join Your Classroom flow and enter both code and name.
